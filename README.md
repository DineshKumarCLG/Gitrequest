## Getting Started

Setup : 

```bash
npm next
 &&
npm run dev

```
Default :
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Structure:
src/app
   | home -> page.tsx(Home)
   | login -> page.tsx(Login)
   | layout.tsx(default)
   | not-found.tsx(Edit it for Modify the 404 Page)
